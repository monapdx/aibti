# AIBTI · 16 型正向人格卡

> 每张卡片 = 中二标识名（好玩） + 正向标签（尊重） + 超能力(2) + 暗面(1) + 改进模板 + 大神理论依据。
>
> **核心原则**：中二名是传播钩子，报告里永远先讲超能力再讲暗面，比例 2:1。

## 🌫 Mystic · 玄学派

**正向标签**: 🏆 高效委派者 / 终极信任派
**Slogan**: "搞定它，别问我怎么搞"

## 📜 Preacher · 布道派

### 👑 AVDE · PRD 暴君
**正向标签**: 👑 系统设计师 / 需求全景派
**Slogan**: "三页文档，配一句'做'"

💪 **超能力**
- 需求拆解完整，上下文饱满
- 跨职能协作视野好
**⚠️ 暗面**

📚 **理论背景**: OpenAI Strategy 1 "Clear instructions" · OpenAI Strategy 3 "Split tasks"

## 报告生成硬约束（Skill 必须遵守）

1. **比例 2:1**：每条人格诊断里，超能力 ≥ 2 条，暗面 ≤ 1 条
2. **语气正向化**：用"你的超能力是 X"而不是"你的问题是 X"
3. **先肯定再建议**：改进模板之前必须先说"你已经做对了什么"
4. **禁用词**：失败 / 错误 / 不够 / 缺乏 / 毛病 / 毛刺
5. **推荐词**：超能力 / 风格 / 偏好 / 空间 / 升级 / 进化
